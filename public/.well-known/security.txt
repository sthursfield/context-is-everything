# Security Disclosure

Context is Everything takes security seriously. If you discover a security vulnerability, please report it responsibly.

# Scope

This security policy applies to:
- www.context-is-everything.com
- All subdomains of context-is-everything.com

# Out of Scope

- Third-party services and integrations
- Issues already publicly disclosed
- Social engineering attacks

# Response Time

We aim to acknowledge security reports within 48 hours and provide a detailed response within 7 days.

Thank you for helping keep Context is Everything secure.
